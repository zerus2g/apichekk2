🚀 API Proxy cho chkr.cc

📋 Thông tin API

Đây là API proxy PHP để kiểm tra thẻ tín dụng thông qua chkr.cc

🔗 URL Format:

https://apichekk2-8aq1.vercel.app/chk?user=CARD_DATA&key=API_KEY

📝 Ví dụ sử dụng:

https://apichekk2-8aq1.vercel.app/chk?user=4154644401584181|02|2027|555&key=khang

⚙️ Tham số:

  • user: Dữ liệu thẻ (format: số_thẻ|tháng|năm|cvv)
  • key: API key (mặc định: khang)
✅ API đã sẵn sàng sử dụng!
